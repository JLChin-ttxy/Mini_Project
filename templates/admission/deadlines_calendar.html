Important Registration Dates & Periods

How Important Dates Are Calculated

Registration periods are calculated consistently each year using the following formula:

  • Semester 1 Registration: Opens 3 months before semester start (October of previous year), closes 2 weeks before semester start (mid-December)
  • Semester 2 Registration: Opens 3 months before semester start (February), closes 2 weeks before semester start (mid-April)
  • Semester 3 Registration: Opens 2 months before semester start (July), closes 1 week before semester start (late August)
  • Application Deadlines: Always 4 weeks before registration opens (to allow time for processing)
  • International Student Deadlines: 2 weeks earlier than local students (to allow time for visa processing)

Note: These dates remain consistent year-to-year, only the year changes. For example, Semester 1 2026 will follow the same pattern as 2025, just shifted by one year.
